:timezone_plug:
> Seleccione su zona horaria en la lista desplegable.
:end

:datetime_map_plug:
> El mapa del mundo destaca la zona horaria actual seleccionada.
>
> Desplácese con el ratón por el mapa y haga clic en el país para el que desea establecer la zona horaria. La zona adecuada se selecciona automáticamente en la lista desplegable.
:end

:use_ntp_plug:
> Seleccione "Sí" para utilizar el Protocolo de Hora de Red para mantener la hora de su servidor de forma precisa.
> Nosotros **recomendamos** el uso de un servidor de tiempo de red, especialmente si usted planea usar Active Directory.
:end

:ntp_server1_plug:
> Este es el servidor NTP primario a utilizar.  Introduzca un FQDN o una dirección IP.
:end

:ntp_server2_plug:
> Este es el servidor NTP alternativo que se utilizará si el servidor NTP 1 no funciona.
:end

:ntp_server3_plug:
> Este es el servidor NTP alternativo que se utilizará si los servidores NTP 1 y 2 no funcionan.
:end

:ntp_server4_plug:
> Este es el servidor NTP alternativo que se utilizará si los servidores NTP 1, 2 y 3 no funcionan.
:end
:current_time_plug:
> Introduzca la hora actual del día.  Utilice el formato AAAA-MM-DD HH:MM:SS. Se muestra en gris cuando se utiliza NTP.
:end
