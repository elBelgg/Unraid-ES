::outgoing_proxy_enable_plug:
> Seleccione el proxy a utilizar. Sólo se podrán seleccionar proxies en línea.
>
> Si su entorno de red requiere un proxy http saliente, defínalo aquí.
>
> Las conexiones salientes desde WebGUI y algunos procesos del sistema utilizarán el proxy http especificado. Las instalaciones y actualizaciones de los contenedores Docker utilizarán el proxy, pero el contenedor en sí no lo hará, ni tampoco las máquinas virtuales.
>
> Para una solución más completa, puede considerar la posibilidad de crear un <u><a href='https://docs.unraid.net/unraid-os/manual/security/vpn/#configuring-vpn-tunneled-access-for-system/' target='_blank'>_(Acceso al Sistema por VPN)_</a></u>.
:end

> Nombre del Proxy Saliente para este Proxy
:end

:outgoing_proxy_url_plug:
> URL del Proxy Saliente para este Proxy. La URL se puede ingresar con o sin credenciales. Las credenciales se analizarán desde la URL.
>
> Si ingresa un usuario y contraseña, se utilizarán como nuevas credenciales.
:end

> Nombre de usuario de Proxy Saliente para este Proxy
:end
